# 初期設定
- Node.js, npmを導入
- npm update
- git clone git@github.com:koganeirail/web.git

# 編集方法
お好みのエディタで ejs ディレクトリ以下を触る。
それ以外は自動生成されるので、直接編集してはいけない

# ビルド方法
- npm run gulp ejs

or

- npm run gulp watch

# 反映方法
- git status
- git add -u
- git commit
- git push
- しばらく待つ or CloudFlareのコントロールパネルからキャッシュクリア
